# Liora — Privacy Policy

Public privacy policy site for the **Liora — Blue Light Filter** Android app.

## Live site

After GitHub Pages is enabled on this repo, the site is live at:

<https://prakash2749.github.io/liora-privacy-policy/>

## Local preview

```bash
python -m http.server 8080
# Then open http://localhost:8080
```

## Files

- `index.html` — the policy itself
- `style.css` — styling that matches the Liora app's design language
- `.nojekyll` — tells GitHub Pages to skip Jekyll and serve files directly
